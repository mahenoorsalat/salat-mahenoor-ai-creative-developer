## Overview

This project focuses on the "art" of the portfolio. It's not just about listing work; it's about creating an atmosphere. Every scroll, hover, and interaction is meticulously choreographed to convey a sense of premium quality and attention to detail.

## Key Features

- **Cinematic Motion**: Heavy use of scroll-triggered animations and smooth parallax effects.
- **Typography Driven**: Bold, impactful typography that guides the user's eye.
- **Dark Mode Aesthetic**: A sleek, dark theme that highlights visual content vividly.
- **Performance Optimized**: Despite the heavy visuals, the site remains performant and responsive.

## Tech Stack

- **Framework**: Next.js
- **Animation**: GSAP (GreenSock Animation Platform)
- **Styling**: SCSS / Tailwind CSS

This portfolio is designed to win awards and clients alike, proving that the web can be as immersive as cinema.
